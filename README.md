# Java 學習紀錄

## 學習目標
- 掌握 Java 程式設計基礎
- 理解物件導向程式設計概念
- 能夠獨立開發小型 Java 應用程式
- 為未來學習 Spring Framework 打好基礎

## 學習大綱

### 1. Java 基礎 (預計 2-3 週)
- [ ] 開發環境設置 (JDK, IDE)
- [ ] 基本語法
- [ ] 變數與資料類型
- [ ] 運算子
- [ ] 控制流程 (if-else, switch)
- [ ] 迴圈 (for, while)
- [ ] 陣列

### 2. 物件導向程式設計 (預計 3-4 週)
- [ ] 類別與物件
- [ ] 封裝
- [ ] 繼承
- [ ] 多型
- [ ] 介面
- [ ] 抽象類別
- [ ] 套件管理

### 3. Java 進階概念 (預計 2-3 週)
- [ ] 例外處理
- [ ] 集合框架 (ArrayList, HashMap 等)
- [ ] 泛型
- [ ] 檔案 I/O
- [ ] 多執行緒基礎

### 4. 實用工具和 API (預計 2 週)
- [ ] 字串處理
- [ ] 日期時間 API
- [ ] Lambda 表達式
- [ ] Stream API
- [ ] Optional 類別

### 5. 資料庫程式設計 (預計 2-3 週)
- [ ] JDBC 基礎
- [ ] SQL 基本操作
- [ ] 資料庫連線
- [ ] CRUD 操作
- [ ] 交易處理
