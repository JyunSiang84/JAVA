## 專案練習
### 1. 階段一 - 基礎語法專案：簡易計算機（第五週完成）
這個專案將幫助您鞏固 Java 基本語法和程式設計概念。計算機需要處理使用者輸入、執行計算，並正確顯示結果。
練習重點：
- 基本運算邏輯實作（加、減、乘、除）
- 流程控制處理使用者輸入
- 方法模組化設計
- 基礎資料驗證

技術需求：
- 使用基本資料型態處理數值
- 實作基礎錯誤檢查
- 模組化的方法設計
- 基本的使用者介面

成果要求：
- 完整的程式碼文件與註解
- 單元測試覆蓋率達 80% 以上
- 使用者操作手冊
- 程式碼品質報告
- 簡單的技術文件說明

### 2. 階段二 - 物件導向專案：銀行帳戶系統（第九週完成）
這個專案將幫助您深入理解物件導向程式設計的核心概念。系統需要管理不同類型的銀行帳戶和交易操作
練習重點：
- 類別和物件的設計與實作
- 帳戶資訊的封裝處理
- 不同帳戶類型的繼承關係
- 交易相關的例外處理
系統功能：
- 支援多種帳戶類型（支票、儲蓄）
- 基本的帳戶操作（存款、提款、轉帳）
- 交易歷史記錄
- 帳戶餘額查詢
成果要求：
- 完整的類別圖與系統架構說明
- 詳細的資料流程文件
- 完善的異常處理機制與文件
- 交易紀錄功能實作
- 測試案例和測試報告
- API 文件
  
3. 階段三 - 進階功能專案（第十二週完成）
這個專案整合了進階的 Java 功能，著重於資料處理和分析。系統需要有效管理學生資料和成績資訊。
練習重點：
- 使用集合框架管理學生資料
- 實作檔案讀寫功能
- 應用 Lambda 和 Stream 進行成績分析
- 泛型設計提升程式碼重用性
系統功能：
- 學生資料的 CRUD 操作
- 成績統計分析（平均、排名、分布）
- 資料匯出匯入功能
- 客製化的成績報表產生
成果要求：
- 系統架構文件
- 資料持久化方案說明
- 效能測試報告
- 使用者介面設計文件
- 完整的測試案例
